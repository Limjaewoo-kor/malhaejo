## 말해조 🗣️
공손한 문장을 AI로 자동 생성해주는 웹 서비스입니다.

### 실행 방법
1. frontend: `npm install && npm start`
2. backend: `uvicorn main:app --reload`

### 환경 변수
- `REACT_APP_API_URL`
- `OPENAI_API_KEY`
